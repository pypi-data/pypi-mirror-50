FSLeyes¶

This is the user documentation for FSLeyes *, the FSL image viewer. This documentation pertains to FSLeyes 0.30.1.

You can download the latest version of FSLeyes from the FSLeyes home page.

If you are getting started with FSLeyes, check out the Quick start page. Otherwise, choose a topic from the list below.
